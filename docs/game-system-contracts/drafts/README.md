# Game System Contract Working Drafts

Status: active draft destination

Use this folder for evolving multi-chat game-system or rules-core contract drafts that should accumulate in one durable working file across ChatGPT and Codex sessions.

Approved file pattern:

`docs/game-system-contracts/drafts/<TOPIC>_WORKING_DRAFT.md`

Use this destination when a long design sequence should continue through a stable repo file instead of many one-off preservation packets.

Completion candidates may use `<TOPIC>_CONTRACT_rev0.1.md` when a working draft has resolved its decisions into a reviewable, versioned contract. The working draft remains the decision record until the contract is accepted.
