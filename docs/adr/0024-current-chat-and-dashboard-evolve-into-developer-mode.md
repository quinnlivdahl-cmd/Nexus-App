# Current chat and dashboard evolve into Developer Mode

The existing chat/dashboard shell does not survive as the normal player interface; spatial play, the Intent Bar, Lattice Ticker, contextual interaction, and tactical HUD replace it. Its useful inspection capabilities evolve into an optional Developer Mode for Context Broker packets, Campaign Director requests and outputs, proposed versus committed state, rules validation, Location assembly, Lattice history, hidden Campaign Threads, and save/export debugging.
