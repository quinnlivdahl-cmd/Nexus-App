# World Timeline is reference, not a forced recap

A Legacy World does not begin each campaign with a mandatory `since last time` recap. The main menu instead exposes a voluntary World Timeline containing the authored setting chronology through the first campaign start and one dated event tick for every completed campaign, with its major validated world changes available as reference. Timeline visibility is separate from character knowledge and Context Broker selection, so an event can be visible to the player without being loaded into every model call or known to every speaker. A future public game might support server-owned shared timelines, but multiplayer canon authority, concurrency, moderation, privacy, rollback, and governance remain outside the local single-player contract.

ADR 0065 restricts timeline ticks to public canon while preserving secret Saga entries for authorized Campaign Director use.
