# Tactical Participation changes only at commit boundaries

Tactical Participation is re-evaluated after committed transactions rather than frozen when Tactical Pressure begins or inferred continuously from presentation frames. A participant joins when committed Game Truth makes an active trigger selector include it, and leaves only when no active trigger selects it and it owns no unresolved timed effect or reaction. This allows consequential movement, awareness, access, hazard, and objective changes to alter participation without mid-transaction churn, retroactive actions, or a second state authority.
