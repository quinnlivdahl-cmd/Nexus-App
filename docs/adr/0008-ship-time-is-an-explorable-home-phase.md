# Ship Time is an explorable home phase

The crew normally returns to its persistent explorable Ship between every Route Node. Recovery, equipment, Field Team selection, Crewmate relationships, recruitment, onboard events, and the next Route Choice occur during Ship Time, with efficient Management Menus available for administration. The Ship is primarily a mobile home base rather than a ship-combat platform or detailed player-controlled repair and upgrade simulation.
