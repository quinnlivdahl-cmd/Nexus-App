# Ship Time is an explorable home phase

The crew normally returns to its persistent explorable Ship between every Route Node. Recovery, equipment, Field Team selection, Crewmate relationships, recruitment, onboard events, and the next Route Choice occur during Ship Time, with efficient Management Menus available for administration. The Ship is primarily a mobile home base rather than a ship-combat platform or detailed player-controlled repair and upgrade simulation.

ADR 0043 supersedes the mandatory Ship return and `Ship Time` name with ownership-neutral Downtime. The accepted between-node activities and the Ship's explorable mobile-home role remain, but Downtime may occur from another suitable staging place. ADR 0044 separately keeps literal Ship Transit as the normal post-choice journey to the next distinct Location.

ADR 0046 sets the activity order: Route Prospects appear at the start of Downtime, the player commits a Route Choice, and only then selects the destination-specific Field Team, loadout, and Ship Assignments.
