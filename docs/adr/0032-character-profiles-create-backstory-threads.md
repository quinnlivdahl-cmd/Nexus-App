# Character Profiles create Backstory Threads

After the preserved Character Build Stack, the player writes a freeform Captain Profile before the campaign premise is generated, while selected Crewmates contribute established Crew Profiles and relevant Archive Memories. Lore-compatible personal history may create light Backstory Threads expressed through dialogue, Points of Interest, Check Opportunities, familiar characters, or minor consequences; they never own a Route Node, create a dedicated questline, become central, rewrite world canon, or grant unvalidated mechanics or possessions. Nexus does not promise fully authored character arcs within a campaign.

ADR 0042 renames the universal protagonist concept to Player Character, so `Captain Profile` generalizes to Player Character Profile. The profile timing and Backstory Thread boundaries remain accepted.
