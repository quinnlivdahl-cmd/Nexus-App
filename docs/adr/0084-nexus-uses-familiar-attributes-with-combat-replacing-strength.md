# Nexus uses familiar Attributes with Combat replacing Strength

Nexus uses Combat, Dexterity, Constitution, Intelligence, Wisdom, and Charisma as its six top-level character Attributes to keep the developing rules and character model immediately legible. Combat occupies the familiar Strength slot and the set supersedes the provisional top-level names Force, Traverse, Vital, Systems, Insight, and Network. This vocabulary decision does not import D&D's resolution rules, classes, derived statistics, or other system assumptions; Nexus Skills, Skill Focuses, Abilities, Lattice-100 resolution, and game-specific boundaries remain native.
