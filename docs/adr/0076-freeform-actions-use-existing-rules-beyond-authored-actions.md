# Freeform Actions use existing rules beyond authored actions

Nexus treats Freeform Action as the general rules family for physically plausible Player Intent that has no exact authored-action match. It is used primarily during Free Movement and resolves by composing existing authoritative state, costs, Checks, Effects, and StateDeltas rather than rejecting creativity or inventing new mechanics, equipment, knowledge, geometry, or permissions. Improvise remains the specific Turn-Based Mode action that opens the Intent Bar and pays the applicable tactical cost to enter a Freeform Action.
