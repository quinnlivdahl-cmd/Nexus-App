# Characters use modular animated 2D World Sprites

Characters remain 2D in the Tilted Top-Down world and share directional animation frameworks for movement, cover, weapons, interactions, and reactions. Modular silhouettes, heads, equipment layers, palettes, and selective signature motions provide identity without requiring every Crewmate to receive a bespoke animation set or individual 3D production.
