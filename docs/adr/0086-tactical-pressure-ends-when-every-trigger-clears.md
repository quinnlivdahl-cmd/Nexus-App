# Tactical Pressure ends when every trigger clears

Tactical Pressure ends at the first post-commit evaluation where committed Game Truth clears every active Tactical Pressure Trigger. The current atomic transaction and already-fired reactions finish before the deterministic mode transition; unused activations and pressure-bound Timing Entries are then cancelled, and the same Location enters Local Aftermath. Victory, defeated opposition, and objective completion have no privileged authority: escape, negotiation, stabilization, or another validated result may clear pressure, while an unresolved hazard or objective can keep it active after hostilities cease.

If a Tactical Pressure Trigger activates after every prior trigger has cleared, it begins a new Tactical Pressure period rather than resuming the closed sequence. Rules Core rebuilds participation from current committed Location truth, every participating actor rolls fresh Initiative, the sequence starts at round 1, and a fresh Surprise Evaluation uses the new trigger and current awareness state.
