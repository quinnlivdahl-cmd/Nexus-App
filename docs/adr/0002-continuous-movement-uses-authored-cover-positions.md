# Continuous movement uses authored cover positions

Nexus uses gridless player-facing movement while retaining an internal navigation representation for pathfinding. Characters preserve their spatial positions when play enters Turn-Based Mode, movement is measured along navigable paths, and deliberate soft alignment to authored Cover Positions provides readable directional protection; an intervening low object does not grant cover to distant characters, while genuinely blocking geometry remains occlusion.
