# Free Movement uses WASD, object clicks, and Automatic Facing

Initial Free Movement is keyboard-and-mouse only: WASD provides normalized eight-direction direct control, the mouse selects world objects and interface targets, and characters automatically face movement, interactions, or chosen targets rather than following the cursor. Clicking an object opens its Context Action Menu, and selecting a proximity-bound action paths the controlled character to an authored Interaction Position. Uncontrolled Field Team members receive only the minimum non-blocking follow behavior needed for play and preserve their actual positions when turns begin; controller support, primary click-to-move control, and elaborate follower logic remain outside the initial scope.
