# Generated dialogue and narration remain unvoiced

Freeform Dialogue, Generated Performance, and Area Introductions remain text-only for the initial product, with no synthetic narrator or generated speech. Audio production instead prioritizes ambience, movement, machinery, alarms, weapons, interface feedback, nonverbal character sounds, and a limited library of authored Tactical Barks, preserving dialogue freedom while avoiding synthetic-voice inconsistency and unsustainable voice scope.
