# Ship conflict does not use a separate combat system

The current game version excludes dedicated Ship combat with separate movement, weapons, shields, hull targeting, initiative, and balance. Ship threats resolve through decisions, Lattice Checks, staffed Ship Systems, Ship Modules, and campaign consequences; a rare enemy Shipboard Incursion uses normal Tactical Pressure inside the persistent Ship Location, while the Field Team boarding another vessel is treated as an ordinary explorable Location rather than a new combat mode.
