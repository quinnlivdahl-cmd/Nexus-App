# Default campaign saves preserve consequences

The normal campaign continuously autosaves committed Game Truth and does not provide routine quickloads to reverse Route Choices, Lattice Rolls, Dialogue Outcomes, or Permanent Loss. An optional reload-friendly setting may permit manual checkpoints, save export/import remains available for portability and disaster recovery, and Crew Archive persistence stays separate so restoring campaign data cannot roll back or duplicate cross-run collection history.

ADR 0095 specializes the day-one player experience as one rolling autosave with Continue, Save & Quit, and export/import rather than manual save slots. It also requires durable Route Node Resolution before the recap and post-node transition can be released.
