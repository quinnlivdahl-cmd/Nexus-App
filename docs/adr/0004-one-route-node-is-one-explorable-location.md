# One Route Node is one explorable Location

A campaign run visits ten Route Nodes, and every Route Choice presents at least three consequential candidate destinations. Each selected Route Node opens exactly one persistent explorable Location of variable size rather than another layer of jump points; Route Node Resolution occurs when the crew departs, Route Pressure advances, and the next choices are revealed. Lightweight Transit Events may occur between nodes without counting toward the ten or requiring full Locations. The campaign path is dynamically directed rather than limited to ten canned Locations, and backtracking depends on campaign conditions rather than being universally available.

ADRs 0043 and 0044 clarify that distinct Route Node Locations may share a broader Region. Literal Ship Transit remains the normal connection, often within one body's orbital Region rather than across distant bodies; only rare same-structure transitions omit open-space travel, and a Transit Event remains optional.

ADR 0051 defines Nodes 8 through 10 as an order-sensitive Climax Set whose three distinct Location identities lock after Node 7 while each full Location remains assembled only when selected.
