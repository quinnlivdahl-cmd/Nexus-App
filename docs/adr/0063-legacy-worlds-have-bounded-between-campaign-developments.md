# Legacy Worlds have bounded between-campaign developments

The first call of the Campaign Opening Pipeline may propose a small bounded set of Interval Developments between the prior Campaign Resolution and the new World Calendar start date. Each development must cite an existing Setting Pillar, faction, unresolved pressure, Campaign Clock, or Legacy Consequence plus an elapsed-time trigger and permitted effect. Deterministic validation applies all changes to a candidate snapshot before checking the Main Campaign Thread, Node 0, chronology, and selected Starting Crew; any invalid component rejects the atomic opening proposal. Interval Developments may evolve or supersede bounded world state but cannot invent unrelated history, erase a Setting Pillar, decide the new campaign outcome, invalidate selected crew, or grant universal character knowledge.

Interval Developments cannot kill archived or retired Crewmates. The simple natural-death baseline is owned by deterministic Biological Age rules.

ADR 0064 keeps those developments out of a mandatory opening recap; material public history is available through the voluntary World Timeline instead.
