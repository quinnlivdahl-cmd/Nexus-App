# Campaigns open with a Director-selected Node 0

After Player Character Creation and Starting Crew selection, the first Campaign Director checkpoint establishes the Main Campaign Thread and selects Node 0 as its playable introduction. Node 0 is outside the ten-node Route Map: it is not selected through Route Prospects and does not count as Route Node 1. When neither the Player Character nor a starting Crewmate is a Captain, Node 0 fiction also provides the crew an eligible campaign Ship by its validated resolution without normally making ship acquisition the node's primary premise. Only afterward does Downtime expose the first Route Prospects and let the player choose Route Node 1.

The Campaign Opening Pipeline uses two calls: the first establishes the Main Campaign Thread, initial Director State, and Node 0 selection; after deterministic validation, the second builds the Node 0 plan using the finalized Player Character, Starting Crew, and starting loadouts. Node 0 therefore has no Route Prospect call or separate deployment-adaptation call.

ADR 0062 adds Legacy World start-date selection to the first existing call after Player Character and Starting Crew selection; it does not add a third opening call or permit chronology to invalidate selected crew.

By Node 0 Resolution, the player must understand the campaign's central problem, why it matters to the group, and why further action is warranted. The deeper cause, antagonist, and full stakes may remain hidden; any settled concealed facts belong to hidden Game Truth, while unresolved alternatives remain bounded Director State and cannot be narrated as established truth.

ADR 0071 places every starting character at level 0 during Node 0 and advances them to level 1 after the first Route Choice but before Node 1 Deployment Preparation.
