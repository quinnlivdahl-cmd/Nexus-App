# Live images illustrate but never define play

Runtime visuals use three distinct lanes: cleaned and approved production assets, live deterministic Location assembly from validated modules, and optional Live Illustrations. Live generation initially supports Route Prospect previews and validated Major Moment Illustrations saved to campaign history; it runs asynchronously with caching and fallback behavior, never blocks play, and cannot define playable geometry, mechanics, character identity, item recognition, or Game Truth.
