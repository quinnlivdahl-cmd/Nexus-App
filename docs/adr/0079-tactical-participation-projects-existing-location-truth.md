# Tactical Participation projects existing Location truth

Active Tactical Pressure Triggers use deterministic selectors to project Tactical Participation from existing Location actors, hazards, and objectives. The projection contains actor references and hazard or objective timing entries while positions, resources, statuses, object state, and other authority remain in committed Location truth; proximity may be one authored selector but never enrolls every nearby entity automatically. This makes participation explicit and testable without duplicating state or recreating an Encounter roster as a competing container.
