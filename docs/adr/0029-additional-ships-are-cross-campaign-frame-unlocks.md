# Additional Ships are cross-campaign frame unlocks

The MVP builds one starter Ship Frame, while the target version may unlock a small Ship Registry through campaign outcomes. A player selects one frame when starting a campaign; frames share the five Ship Systems and seven-character roster model but may vary in modular interior layout, visual identity, slot biases, starting capability, and one signature support feature, while installed Ship Modules and Ship Conditions reset with the run rather than creating fleet management.

ADR 0042 supersedes universal start-of-campaign frame selection. Direct selection applies when the Player Character is a Captain; a Captain starting Crewmate brings their profile-bound Ship, and otherwise the Campaign Director provides an eligible frame through Node 0. The shared-system, Ship Registry, variation, and reset boundaries remain accepted.
