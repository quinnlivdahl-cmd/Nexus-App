# Irreversible Captain loss ends the campaign

The existing recovery baseline survives: reaching zero creates a visible Downed or Disabled countdown, rescue and stabilization remain possible, and a failed countdown creates a critical state rather than immediate deletion. Only Permanent Loss of the Captain after applicable recovery and revival avenues are exhausted ends the current campaign run and records its history in the Crew Archive.
