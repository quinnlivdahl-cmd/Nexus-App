# Lattice Ticker placement is deferred

The Lattice Ticker remains a compact, nonmodal, persistent result surface that never pauses or blocks spatial play, but this decision does not assign it to a corner, the Intent Bar, or any other exact screen position. Exact placement and HUD composition require later UI work with representative spatial play. This refines the placement language in [ADR-0011](0011-lattice-100-and-visible-rolls-remain-core.md) while preserving its decisions that Lattice rolls remain visible and use a nonmodal ticker.
