# Downtime is location-neutral between Route Nodes

Downtime replaces Ship Time as the ownership-neutral phase after Node 0 and between Route Nodes, retaining recovery, equipment, Field Team selection, relationships, recruitment, Ship preparation, and Route Choice without requiring a physical return to the Ship. It may occur aboard the Ship or from another suitable staging place and is distinct from the Transit that normally follows a Route Choice. Consecutive Route Nodes may share a broader Region but must use distinct playable Locations; revisiting an existing Location is backtracking rather than a new numbered Route Node.
