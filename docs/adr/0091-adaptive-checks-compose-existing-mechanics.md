# Adaptive Checks compose existing mechanics

The Campaign Director may respond like a good DM by proposing a situation-specific Check for a grounded player action that authored options did not anticipate, including a proposed approach, Skill, stakes, and outcome meanings. Rules Core must map the proposal to existing Check families, Target Score rules, result bands, Effects, and StateDeltas and validate the complete Check against current Game Truth before any roll. The proposal cannot create a new mechanic, state type, object, item, ability, permission, or effect, and Generated Performance may describe only the validated committed result. This specializes [ADR-0076](0076-freeform-actions-use-existing-rules-beyond-authored-actions.md) without creating a second resolution system.
