# Campaigns start at level 0 and advance before Node 1

Every newly created Player Character and selected Starting Crewmate begins a campaign at level 0 and plays Node 0 at that level. Level 0 is a complete starting build with authored foundations, assigned traits, starting abilities or populated trees, and a usable loadout rather than an empty character. The player chooses the Player Character's Starting Loadout during Player Character Creation before Campaign Opening call 1; each selected Starting Crewmate arrives with a predetermined profile-bound Starting Loadout rather than asking the player to rebuild every Crewmate before play. Both opening calls therefore receive a complete starting deployment, and Node 0 has no additional preparation step between them. After Node 0 Resolution, the player reviews and confirms the first Route Choice; before Node 1 Deployment Preparation, every current campaign character advances to level 1. The same cadence applies throughout the campaign: after Route Node N-1 and confirmation of Route Node N, current campaign characters advance to Level N before that node's Deployment Preparation, so Route Nodes 1 through 10 are played at Levels 1 through 10 respectively. Milestone levels add their defined extra benefits without replacing the ordinary level advancement. Former Player Character Conversion and all returning Crewmates use the same level-0 reset. Exact advancement benefits and milestone cadence remain progression-design work, while advancement timing lets the player prepare the newly leveled crew for a known destination.
