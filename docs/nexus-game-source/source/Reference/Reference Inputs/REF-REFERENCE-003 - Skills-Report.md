**Recommended implementation sketch**

- Each turn: **3 actions + 1 reaction + 1 free micro-interaction**.
- Spending **1 action to Move** grants your current **Move Rating** in MP.
- Characters may take **Move** multiple times if they want to invest their entire turn into mobility.
- Some features grant **bonus MP**, **stored MP**, or **MP refunds**, but those should usually be conditional.
- Stances should prepare specific reactions rather than granting broad universal opportunity attacks.
- Once/round rider caps should limit reaction spam and prevent support loops from becoming mandatory.
