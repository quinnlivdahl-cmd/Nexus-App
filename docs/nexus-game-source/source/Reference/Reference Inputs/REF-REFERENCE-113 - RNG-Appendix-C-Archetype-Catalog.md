# Nexus RNG Appendix C: Archetype Catalog
Revision: 1.0  
Status: Draft  
Last Updated: 2026-05-02

## Purpose

This appendix defines non-canon test archetypes for RNG and balance work.

These archetypes are not final Nexus classes. They are stress-test models used to check whether the RNG and combat system can support varied playstyles without making any option dominant or unplayable.

## Canon Control

Archetypes in this appendix are test models only. They do not define official classes, backgrounds, species, recruits, or abilities.

A playstyle becomes canon only when approved into the relevant character, class, recruit, or combat document.

## Archetype Template

Each archetype should eventually define:

| Field | Description |
|---|---|
| Name | Test archetype name. |
| Core fantasy | What it should feel like. |
| Primary output | Damage, control, defense, healing, scouting, mobility, etc. |
| RNG dependence | Accuracy, many rolls, few rolls, rerolls, degrees of success, etc. |
| Resource dependence | Heat, focus, ammo, cooldown, stress, charge, etc. |
| Strength profiles | Enemy or mission types where it should shine. |
| Weakness profiles | Enemy or mission types where it should struggle. |
| Balance risk | How it might become dominant or useless. |
| Test metrics | What should be measured. |

## Draft Test Archetypes

### Precision Striker

Core fantasy: a specialist who relies on a small number of high-impact actions.

Expected strengths:

- single elite targets
- exposed priority targets
- tactical setup turns

Expected weaknesses:

- swarms
- decoys
- heavy cover
- shielded enemies
- long attrition if resources are limited

Balance risk:

If accuracy and damage scale together, the archetype may become too reliable and too explosive.

### Volume Attacker

Core fantasy: a combatant who makes many smaller attacks or repeated attempts.

Expected strengths:

- low-defense enemies
- shield stripping
- finishing wounded targets
- triggering on-hit effects

Expected weaknesses:

- armor that reduces each hit
- retaliation effects
- ammunition pressure
- enemies that punish repeated rolls

Balance risk:

If each hit can trigger strong secondary effects, the archetype may scale faster than intended.

### Durable Defender

Core fantasy: a tank, bodyguard, or armored anchor.

Expected strengths:

- attrition fights
- chokepoints
- escort objectives
- enemies with predictable attacks

Expected weaknesses:

- bypass damage
- hacking
- mobility objectives
- effects that ignore armor

Balance risk:

If defense also improves control, damage, or action economy, the defender may become universally optimal.

### Evasive Skirmisher

Core fantasy: a mobile character who survives by movement, evasion, and positioning.

Expected strengths:

- objective play
- flanking
- hit-and-run encounters
- enemies with poor tracking

Expected weaknesses:

- area effects
- forced movement
- accuracy-independent effects
- confined spaces

Balance risk:

If mobility prevents too much enemy interaction, the archetype may invalidate encounter pressure.

### Controller / Disabler

Core fantasy: a character who reduces enemy options instead of dealing primary damage.

Expected strengths:

- dangerous elites
- setup-heavy enemies
- mixed encounters
- party synergy

Expected weaknesses:

- control-resistant bosses
- swarm encounters
- time-limited damage races

Balance risk:

If control is too reliable, it can remove the game from enemies. If too unreliable, it feels useless.

### Support / Stabilizer

Core fantasy: a character who protects, repairs, heals, buffs, or enables others.

Expected strengths:

- long missions
- attrition sequences
- protecting fragile allies
- amplifying specialists

Expected weaknesses:

- solo checks
- burst-damage races
- anti-support enemies

Balance risk:

Support can become mandatory if encounter damage assumes it, or weak if the game rewards only damage.

### Drone / Summon Commander

Core fantasy: a character whose power is distributed across drones, pets, programs, or controlled agents.

Expected strengths:

- action coverage
- scouting
- multi-objective rooms
- sacrificial tactics

Expected weaknesses:

- area damage
- command disruption
- hacking
- maintenance or resource pressure

Balance risk:

Extra bodies can quietly break action economy even if each unit is weak.

### High-Variance Gambler

Core fantasy: a build that accepts risk for exceptional upside.

Expected strengths:

- desperate situations
- burst windows
- risky technology
- dramatic clutch moments

Expected weaknesses:

- long attrition
- reliability checks
- missions where failure cascades

Balance risk:

If downside is avoidable, it becomes simply overpowered. If downside is too punishing, it becomes unplayable.

### Hacker / System Intruder

Core fantasy: a character who attacks systems, sensors, networks, robots, locks, signals, and infrastructure.

Expected strengths:

- robotic enemies
- electronic defenses
- locked environments
- information warfare

Expected weaknesses:

- biological enemies
- hardened systems
- signal-dead zones
- physical rush pressure

Balance risk:

If everything has systems to exploit, the hacker becomes universal. If too many things are immune, the archetype disappears.

## Archetype Coverage Checklist

When testing a ruleset, ask:

1. Does each archetype have at least one favorable matchup?
2. Does each archetype have at least one meaningful weakness?
3. Does any archetype win across most profiles?
4. Does any archetype depend on GM charity to matter?
5. Does the RNG system support both reliable specialists and volatile risk-takers?
6. Does the action economy support support, drones, and control without letting them dominate?

## Update Triggers

Update this appendix when:

- a new class, role, origin, recruit type, or build path is proposed
- a major playstyle becomes unlikely or impossible under current rules
- a playtest reveals a missing archetype
- simulations show a dominant or dead archetype
- character creation or recruit generation becomes Draft Active
