## Promotion Decisions

Default handling:

- Keep reference reports intact when they provide useful context.
- Promote only the specific useful passage or finding, not the whole report.
- Route promoted findings to the owning domain.
- Keep PDF/source-material indexes as pointers unless extraction is requested.
- Preserve RNG appendices as tuning evidence until Skills/Data/Automation need specific tables or procedures.
- Preserve lore guidance as evidence unless `Lore` accepts it as canon or deferred seed.

## Known Risks

- Reference summary IDs are evidence surfaces, not game canon.
- Research material may contain valuable ghost content that was never promoted.
- Reports can be too dense for direct review; future mining should produce compact comparison packets.

## Preservation Rule

Cleanup requires content coverage verification and explicit user approval.
