# Research Findings Index

## Purpose

This index helps locate reference material by topic and routes it to the correct rebuilt source domain. It is a findability and routing document, not canon authority.

## Source material index

### Systems / mechanics

- `REF-SURVEY-001`: comparative system and mechanics survey; useful for `Core`, `Combat`, `Skills`, `Characters`, `Equipment`, and `Play Aids`.
- `REF-REFERENCE-003`: skill/progression/focus report; useful for `Skills`, `Characters`, and `Equipment` cross-checks.
- `REF-REFERENCE-110` through `REF-REFERENCE-118` plus `REF-RNG-CHANGELOG-001`: RNG appendices; useful for `Skills`, `Content`, and future `Data` workbooks.

### Lore / setting / tone

- `REF-RUNBOOK-102`: reference guidance for world state and plausibility; useful for `Lore`.
- `REF-SCIFI-SEED-001`: broad idea compendium; useful for `Lore`, `Art`, and deferred drafting, but not canon by default.
- `REF-RNG-BACKBONE-001`: broad balance/research input; use as reference only.

### Admin / runtime

- `REF-REFERENCE-101`: research report on AI/runtime/manual behaviors; useful for `Admin`, `Modes`, `Automation`, and `Dashboards` only if promoted.

### PDF source materials

- `REF-130 - PDF_Source_Materials_Index.md`: lists PDFs that were found but not converted/promoted in this pass.

## Search/use guidance

When a future mode needs reference support:
1. Search this package first by topic.
2. Identify whether the material is reference-only or already promoted.
3. If useful but unpromoted, route it to Draft/Steward for source integration.
4. Do not silently override active source docs.
