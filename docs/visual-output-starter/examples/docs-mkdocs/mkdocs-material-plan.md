# MkDocs Material Example: Small Docs Site

Use this lane when Codex needs a navigable documentation site rather than one Markdown file.

## Minimal Structure

```text
docs-site/
  mkdocs.yml
  docs/
    index.md
    getting-started.md
    reference.md
```

## Good Uses

- local operator handbook;
- reviewable project guide;
- exported knowledge pack;
- visual documentation for a workflow system.

## First Pass Rule

Start with normal Markdown in the repo. Add MkDocs only when navigation, search, or export quality materially improves the deliverable.
