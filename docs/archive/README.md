# Nexus Historical Archive

Status: searchable historical reference; excluded from default current-state retrieval

This folder preserves superseded Nexus project controls, bridge baselines, decision syntheses, migration checkpoints, and workflow candidates after their replacements became active.

Archive material is intentionally retrievable for provenance, audits, or reconstruction. It does not control current game design, project workflow, planning, or ChatGPT behavior. Start with `AGENTS.md`, `CONTEXT-MAP.md`, live GitHub Issues, canonical source, and accepted ADRs instead.

`ARCHIVE-INDEX.json` records each file's former active path and current replacement. Git history preserves the move and earlier revisions.

No canonical game-source document is archived merely because it uses older terminology. Current game rules, concepts, and lore remain under `docs/nexus-game-source/source` and keep their document-level retrieval classification.
